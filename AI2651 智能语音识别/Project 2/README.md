## 概述

在给定10小时的中文数据集上，利用 Kaldi [或 PyTorch (可选)]，搭建完整的语音识别系统，包括：

- 数据的处理
- 特征的提取
- 模型训练

  - 选做：基于深度学习的模型训练
- 测试集上的识别解码

2. 不可以使用额外数据进行训练
